# Modern Login Page

A modern and responsive login page built using HTML and Tailwind CSS. The design features a glassmorphism-inspired interface with a background image, social login buttons, and a clean user experience.

## Features

- Responsive design
- Glassmorphism-inspired UI
- Background image with overlay
- Email and password login form
- Password visibility icon
- Forgot Password link
- Social login buttons (Google, GitHub, Facebook)
- Clean and modern layout
- Built with Tailwind CSS

## Technologies Used

- HTML5
- Tailwind CSS
- Font Awesome

## Purpose

This project was created to practice building modern authentication interfaces using HTML and Tailwind CSS.
